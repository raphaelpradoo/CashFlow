## 📌 Sobre o projeto

Este repositório contém a solução de um projeto prático de um **Gerenciador de Financeiro** desenvolvido durante a **Formação C# da Rocketseat**, com o objetivo de aplicar, na prática, os conceitos estudados ao longo do curso. O projeto propõe aplicar conteúdo visto em aula.

---

## 🎯 Objetivos de Aprendizado

- Aprender sobre **Validações e Exceções**
- Aprender sobre **Testes Unitários**
- Aprender sobre **Bancos de Dados**
- Gerar relatórios em **PDF e Excel**

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C#
- **Plataforma:** .NET
- **IDE:** Visual Studio
- **Tipo de Aplicação:** API REST
- **Validação:** FluentValidation
- **Banco de Dados:** MySQL

---

## ▶️ Como Executar

- Clone o repositório:

git clone https://github.com/raphaelpradoo/CashFlow

- Abra o arquivo .slnx no Visual Studio
- Abra o terminal de execução dentro da pasta do projeto e execute os seguintes comandos:<br>
`dotnet add package Swashbuckle.AspNetCore`<br>
`dotnet restore`<br>
`dotnet build`<br>
